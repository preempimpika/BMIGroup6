เขียน Code ในส่วนของ Application Resources ที่เป็นการเปลี่ยนระดับสีของค่า BMI หลังจากคำนวณออกมาว่าอยู่ในเกณฑ์อะไร,
หา Code สีเพื่อนำมาตกแต่งแอปพลิเคชันให้ดูสวยงามยิ่งขึ้น
